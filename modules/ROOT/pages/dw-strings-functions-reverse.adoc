[[reverse1]]
== reverse&#40;text: String&#41;: String

Returns the reverse sequence of characters of the given `text`

_Introduced in DataWeave 2.3.0. Supported by Mule 4.3 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The text to be reverse
|===

=== Example

This example shows how the `reverse` behaves under different inputs.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
import * from dw::core::Strings
output application/json
---
 {
     a: reverse("Mariano"),
     b: reverse(null),
     c: reverse("")
 }

----

==== Output

[source,Json,linenums]
----
{
  "a": "onairaM",
  "b": null,
  "c": ""
}
----
