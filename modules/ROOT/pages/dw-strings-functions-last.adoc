[[last1]]
== last&#40;text: String, amount: Number&#41;: String

Returns the last `amount` characters of the string.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string.
| `amount` | The amount of characters to get.
|===

=== Example

This example shows how the `last` function takes the last 6 characters from
a string.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import last from dw::core::Strings
output application/json
---
"hello world!" last 6

----

==== Output

[source,Json,linenums]
----
"world!"
----
