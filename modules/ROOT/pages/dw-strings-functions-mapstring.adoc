[[mapstring1]]
== mapString&#40;@StreamCapable text: String, mapper: &#40;character: String, index: Number&#41; &#45;&#62; String&#41;: String

Maps a function over every character of the string.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The text to map to.
| `mapper` | The function to apply.
|===

=== Example

This example shows how the `mapString` can be used to redact important data.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import mapString from dw::core::Strings
output application/json
---
"Account balance: \$234" mapString if (isNumeric($)) "~" else $

----

==== Output

[source,Json,linenums]
----
"Account balance: $~~~"
----
