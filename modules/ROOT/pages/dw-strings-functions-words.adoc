[[words1]]
== words&#40;text: String&#41;: Array<String&#62;

Returns the individual words of a given string.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string to get words from
|===

=== Example

This example shows how a string is divided by `words`.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import words from dw::core::Strings
output application/json
---
words("hello world\nhere\t\t\tdata-weave")

----

==== Output

[source,Json,linenums]
----
["hello", "world", "here", "data-weave"]
----
