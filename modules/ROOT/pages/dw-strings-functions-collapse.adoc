[[collapse1]]
== collapse&#40;text: String&#41;: Array<String&#62;

Collapses the string into substrings of equal characters.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string to collapse
|===

=== Example

This example shows how the function collapses the same characters.
Notice that whitespace (" ") is a character.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import collapse from dw::core::Strings
output application/json
---
collapse("a  b bbb a")

----

==== Output

[source,Json,linenums]
----
["a", "  ", "b", " ", "bbb", " ", "a"]
----
