[[first1]]
== first&#40;text: String, amount: Number&#41;: String

Returns the first `amount` characters of the string.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string.
| `amount` | The amount of characters to get.
|===

=== Example

This example shows how the `first` function takes the first 5 characters from
a string.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import first from dw::core::Strings
output application/json
---
"hello world!" first 5

----

==== Output

[source,Json,linenums]
----
"hello"
----
