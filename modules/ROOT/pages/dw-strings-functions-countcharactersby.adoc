[[countcharactersby1]]
== countCharactersBy&#40;text: String, predicate: &#40;character: String&#41; &#45;&#62; Boolean&#41;: Number

Count the number of times `predicate` returns true for each
characters of `text`.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The text to look in.
| `predicate` | The predicate to apply to each character.
|===

=== Example

This example shows how `countCharactersBy` can be used to count for digits.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import countCharactersBy from dw::core::Strings
output application/json
---
"42 = 11 * 2 + 20" countCharactersBy isNumeric($)

----

==== Output

[source,Json,linenums]
----
7
----
