[[substringby1]]
== substringBy&#40;text: String, predicate: &#40;character: String, index: Number&#41; &#45;&#62; Boolean&#41;: Array<String&#62;

Splits the string at every character where the `predicate` returns true.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string to split.
| `predicate` | The predicate that tests each character.
|===

=== Example

This example shows how the `substringBy` can split a string on a set of given
characters.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import substringBy from dw::core::Strings
output application/json
---
"hello~world=here_data-weave" substringBy $ == "~" or $ == "=" or $ == "_"

----

==== Output

[source,Json,linenums]
----
["hello", "world", "here", "data-weave"]
----
