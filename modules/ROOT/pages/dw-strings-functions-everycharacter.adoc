[[everycharacter1]]
== everyCharacter&#40;text: String, condition: &#40;character: String&#41; &#45;&#62; Boolean&#41;: Boolean

Check whether the condition is valid for every character in `text`

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string to test.
| `condition` | The condition to test it with.
|===

=== Example

This example shows how `everyCharacter` can be used to check
if a string is composed of only digits and spaces.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import everyCharacter from dw::core::Strings
output application/json
---
"12 34  56" everyCharacter $ == " " or isNumeric($)

----

==== Output

[source,Json,linenums]
----
true
----
