[[lines1]]
== lines&#40;text: String&#41;: Array<String&#62;

Returns the individual lines of a given string.

_Introduced in DataWeave 2.4.0. Supported by Mule 4.4 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The string to split into lines
|===

=== Example

This example shows how a string is divided by `lines`.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import lines from dw::core::Strings
output application/json
---
lines("hello world\n\nhere   data-weave")

----

==== Output

[source,Json,linenums]
----
["hello world", "", "here   data-weave"]
----
